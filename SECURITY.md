# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in grome, please report it privately by emailing **leeronalda@gmail.com**.

Please include:
- A description of the issue
- Steps to reproduce
- Affected versions
- Any suggested mitigation

Do not open a public GitHub issue for security-related reports. You will receive an acknowledgement within 72 hours.

## Supported Versions

Only the latest minor release receives security fixes.
